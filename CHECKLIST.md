# Setup Checklist - PolyScores Kenya

Use this checklist to track your setup progress.

---

## Prerequisites

- [ ] JDK 17 installed
- [ ] Android Studio installed (latest version)
- [ ] Google account created

---

## Firebase Setup

- [ ] Firebase project created at https://console.firebase.google.com/
- [ ] Android app registered with package name `com.polyscores.kenya`
- [ ] `google-services.json` downloaded
- [ ] `google-services.json` placed in `app/` folder
- [ ] Firestore Database created (test mode)
- [ ] Authentication enabled (Email/Password)
- [ ] Firestore security rules updated

---

## Firestore Data

- [ ] `teams` collection created
- [ ] `leagues` collection created
- [ ] `matches` collection created
- [ ] `admins` collection created
- [ ] Sample team added
- [ ] Sample match added

---

## Android Studio Setup

- [ ] Project opened in Android Studio
- [ ] Gradle sync completed successfully
- [ ] No build errors
- [ ] Android Virtual Device (emulator) created
- [ ] App builds successfully (Build → Make Project)

---

## Running the App

- [ ] Emulator started
- [ ] App installed on emulator
- [ ] App launches without crashes
- [ ] Bottom navigation works
- [ ] Data loads from Firebase

---

## Testing Features

- [ ] Home screen displays matches
- [ ] Matches screen shows all matches
- [ ] Standings screen shows table
- [ ] Teams screen shows team grid
- [ ] Settings screen loads
- [ ] Admin panel accessible
- [ ] Score update dialog works
- [ ] Match status can be updated

---

## Final Checks

- [ ] Internet connection working
- [ ] Firebase data syncing
- [ ] No errors in Logcat
- [ ] App responsive on different screen sizes

---

## Optional Enhancements

- [ ] Custom app icon added
- [ ] Brand colors customized
- [ ] Multiple teams added
- [ ] Multiple matches added
- [ ] Admin user created in Authentication
- [ ] Push notifications tested
